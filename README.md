# 终审日 / Breaking Rules

> 「规则就是用来打破的。」

**《终审日》** 是一款以「规则即牢笼」为主题的 **Game Jam 横版动作 Boss Rush**，使用 **Godot 4.7（Mono / C#）** 开发。

你是一名被囚禁在「规则之塔」底层的囚徒，坚信这场审判并不公正。牢笼如塔，将你禁锢在最底层——要挣脱，就必须一层层向上，逐一质疑每层法官颁布的「规则」。从最底层的「初审官」开始，帮他打破这座牢笼。

---

## 玩法

- BOSS 会不断颁布**规则**：`禁跳` / `禁武` / `限速`，并会升级为**全图生效**或**跟随你移动**。
- 走到规则区，**长按 Q** 划除规则，将其反转成弹簧，并开启**真空期**（攻击 ×3、移速 ×2）。
- 击败每一层法官，向上一层发起挑战；打穿最终 BOSS「终裁者」后开启**无尽模式**。

## 操作

| 操作 | 按键 |
|------|------|
| 移动 | `A` / `D`（部分规则下左右反转） |
| 跳跃 | `W` / `↑` |
| 攻击 Boss | `J` / `空格` |
| 划除规则 | 长按 `Q` 约 1 秒（期间按任意其他键或松手取消，不进入冷却；完成进 5 秒冷却） |
| 防御 | 按住 `S`（BOSS 攻击前身体后方红色描边光环闪烁预警；防御中完全抵挡 BOSS 伤害，但不可移动 / 攻击） |
| 技能 1 | 毁灭直线（消耗 1 技能点，10s 冷却） |
| 技能 2 | 八向射线（消耗 1 技能点，8s 冷却） |
| 技能 3 | 青色护盾（14s 冷却，激活 3 秒无敌，可抵挡投技） |
| 技能 4 | 自我治愈（18s 冷却，回复 2 点生命） |
| 重开 | `R` |
| 下一关 / 无尽 | `Enter` / `N` |
| 返回标题 | `T` |

技能点由地图随机掉落的**技能宝珠**拾取累积；规则条文可能全图生效或跟随你（标注【全图】/【跟随】），走到消除区长按 Q 划除即可。

> 本游戏的操作说明也是规则，但你不需要遵守它们。

## 技术栈

- 引擎：**Godot 4.7 stable（Mono）**
- 语言：**C# (.NET 8)**，SDK `Godot.NET.Sdk 4.7.0`
- 美术 / 音频：全部使用占位资源（`Util.Square` 程序化生成方块贴图），纯视觉 TileMap 叠加保留的 `StaticBody2D` 碰撞体
- 架构：Autoload 单例（`RuleManager` / `RunState`）+ 信号驱动解耦 + 运行时 `Instance` Boss

## 运行与构建

### 运行（开发）

1. 使用 **Godot 4.7+（Mono 版本）** 打开本项目根目录。
2. 首次打开会自动生成 `Breaking Rules.csproj` / `.sln`。
3. 按 `F5` 运行（标题界面 → 开始游戏）。

### 构建（校验 C# 编译）

```bash
dotnet build "Breaking Rules.csproj"
```

### 导出

项目已配置 `export_presets.cfg`（Windows 导出）。通过 Godot 编辑器 **Project → Export** 生成可执行文件到 `Dist/`（已被 `.gitignore` 忽略，不纳入版本库）。
